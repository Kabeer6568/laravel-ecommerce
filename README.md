# Laravel E-commerce Backend Project

A simple e-commerce application built with Laravel to demonstrate backend development skills including database design, session-based cart handling, checkout flow, and admin CRUD operations.

## Features

- Product listing and product detail pages
- Featured products on homepage
- Session-based shopping cart
- Checkout flow (no real payment gateway)
- Order and order items storage
- Admin panel for managing products
- Input validation and error handling

## Tech Stack

- Laravel
- MySQL
- Blade Templates
- Session handling
- Eloquent ORM

## Project Purpose

This project was built as a learning and portfolio project to practice real-world backend concepts such as:
- MVC architecture
- Business logic implementation
- State management using sessions
- Clean controller and route structure
